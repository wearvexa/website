# Vexa Store

Official website of **Vexa**, a modern fashion and streetwear brand built with **Next.js**.

Vexa is focused on delivering a fast, responsive, and premium shopping experience across all devices.

---

## 🚀 Tech Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
- App Router
- ESLint

---

## 📦 Getting Started

Clone the repository:

```bash
git clone <repository-url>
cd vexa-store
```

Install dependencies:

```bash
npm install

# or

pnpm install

# or

yarn

# or

bun install
```

Start the development server:

```bash
npm run dev

# or

pnpm dev

# or

yarn dev

# or

bun dev
```

Open your browser and visit:

```
http://localhost:3000
```

## 🌍 Environment Variables

Create a `.env.local` file in the project root.

Example:

```env
NEXT_PUBLIC_API_URL=http://localhost:8000/api
```
